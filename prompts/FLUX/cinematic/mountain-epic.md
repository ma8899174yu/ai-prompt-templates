# FLUX 史诗山景

## 效果描述

生成震撼人心的史诗级山景风光：巍峨山峰、晨曦云海、气势磅礴。适合壁纸、海报、旅游宣传等用途。

## 适用场景

- 壁纸设计
- 旅行海报
- 纪录片开场
- 社交媒体封面
- 品牌视觉背景

## 参数建议

| 参数 | 值 |
|:-----|:---|
| Steps | 30–40 |
| CFG | 2.0–3.5 |
| Sampler | dpmpp_2m |
| 分辨率 | 1792×768 (21:9) 或 1536×1024 (3:2) |
| 建议比例 | 21:9（电影宽屏） |
