# 自然语言 - 创意头像

## 效果描述

与众不同的头像设计，既保留个人特征又有艺术美感。适合社交媒体头像、个人品牌视觉。

## 适用场景

- 微信头像
- 小红书头像
- B站头像
- LinkedIn 头像
- 各类社交平台头像

## 正向提示词

```
生成一张个人微信头像。
要求：
- 真人照片风格，但稍微有艺术感
- 背景是温暖的日落天空
- 人物侧脸，望向远方
- 光线是 golden hour 的金色光
- 发丝清晰可见，边缘有光晕
- 整体氛围是自由、浪漫、文艺
- 适合做社交媒体头像
```

## 负面提示词

```
卡通、动漫、丑、变形、比例不对、
多余的手指、模糊、低分辨率、
有文字水印、过度美颜、塑料感、低质量
```
