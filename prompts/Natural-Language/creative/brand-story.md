# 自然语言 - 品牌故事

## 效果描述

如同品牌大片，展现生活方式与产品调性的完美融合。适合独立设计师品牌、初创品牌视觉内容。

## 适用场景

- 品牌官网首图
- Instagram 品牌账号
- 品牌宣传册
- 电商详情页
- PR 礼盒包装

## 正向提示词

```
请为我的独立设计师品牌创作一张视觉图。

场景描述：
在一个阳光透过大窗户洒进来的 loft 工作室里，
一位穿着我们最新系列服装的模特站在窗边，
手里拿着一杯咖啡，神态放松而自信。
背景是极简的工业风装修，有原木色家具和绿植。
整体色调是莫兰迪色系，温暖而高级。
阳光在模特身上形成漂亮的光影轮廓。

风格要求：
- 接近 Vogue 杂志的审美
- 文艺、克制、有故事感
- 不要过度后期，保持自然
- 8K 分辨率，商业可用
```

## 负面提示词

```
过度修图、塑料感、网红风格、美颜过度、
低俗、过曝、构图杂乱、
卡通、动漫、不真实、低质量
```
