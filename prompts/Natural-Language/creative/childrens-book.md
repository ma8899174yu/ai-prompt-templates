# 自然语言 - 儿童绘本插画

## 效果描述

温馨可爱的儿童绘本插画风格，适合讲睡前故事。颜色温暖柔和，线条圆润可爱，适合 3-6 岁儿童。

## 适用场景

- 儿童绘本创作
- 教育内容配图
- 亲子账号素材
- 儿童 App 界面
- 生日派对邀请函

## 正向提示词

```
画一个温暖的睡前故事场景。

画面内容：
- 一只可爱的小兔子穿着蓝色的睡衣
- 它抱着一个胡萝卜形状的毛绒玩具
- 坐在一张软软的小床上
- 窗外有月亮和星星
- 床头有一盏发出暖暖黄光的小夜灯
- 房间里有很多很多书

风格要求：
- 儿童绘本插画风格
- 水彩画的质感
- 颜色温暖柔和
- 线条圆润可爱
- 适合 3-6 岁儿童
- 无任何不适内容
```
