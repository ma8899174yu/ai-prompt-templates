# 自然语言 - 技术解析图

## 效果描述

专业的信息图表设计，适合科技博客和演示文稿。用自然语言描述复杂的技术概念，AI 自动理解并生成高质量可视化图像。

## 适用场景

- 技术博客配图
- 商业 PPT
- 教学视频封面
- 白皮书插图
- 产品说明书

## 正向提示词

```
创建一个解释区块链技术的数据可视化图。

要求：
- 信息图风格，清晰易懂
- 使用蓝色系渐变作为主色调
- 显示一个区块链条，每个区块包含数据
- 中心人物连接多个区块，表示分布式网络
- 使用简洁的图标和线条
- 背景是深色科技风
- 包含"Blockchain"和"分布式账本"文字
- 3D渲染风格，未来科技感
```

## 负面提示词

```
手绘风格、潦草、混乱、低质量、低分辨率、
```
