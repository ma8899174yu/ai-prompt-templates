# 自然语言 - 产品场景展示

## 效果描述

简洁优雅的产品展示图，符合电商平台主图标准。适合各类电商产品的商业摄影风格图像。

## 适用场景

- 电商主图
- 产品官网
- 社交媒体推广
- App Store 截图

## 正向提示词

```
请生成一张高端无线耳机的产品展示图。
要求：
- 纯白背景，专业影棚灯光
- 耳机悬浮在画面中央
- 极简风格，突出产品质感
- 光线柔和，有轻微阴影
- 高端科技感，如同苹果官网风格
- 分辨率 8K
```

## 负面提示词

```
杂乱的背景、阴影过重、颜色失真、
低质量、模糊、有水印、有文字
```
