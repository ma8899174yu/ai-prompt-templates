# 自然语言 - 短视频封面

## 效果描述

高点击率的短视频封面，符合平台推荐算法偏好。用自然语言描述吸引眼球的封面设计，AI 自动生成具有视觉冲击力的图像。

## 适用场景

- B站视频封面
- 抖音/快手短视频
- YouTube 缩略图
- 小红书笔记封面
- 公众号文章头图

## 正向提示词

```
设计一个B站科技区视频的封面图。

要求：
- 背景是纯色渐变，从深蓝到紫色
- 画面主体是一个3D风格的手机模型
- 手机屏幕上显示AI对话界面
- 标题文字"AI觉醒，人类何去何从？"
- 整体风格赛博朋克，有霓虹光效
- 有轻微的故障艺术(glitch)元素
- 震撼、有冲击力、让人想点击
```

## 负面提示词

```
模糊、低质量、低分辨率、水印、文字错位、
过于普通、无视觉焦点、
色情、违规内容
```
